# cursor_cloud_agent_test

A sandbox repository for testing Cursor Cloud Agent workflows.

## Overview

This repo is used to experiment with and validate Cursor Cloud Agent
capabilities, such as branching, committing, pushing, and opening pull
requests automatically.

## Structure

- `README.md` — this file.
- `AGENTS.md` — guidance and environment setup notes for Cloud Agents.
- `.gitignore` — ignored local environment files.

## Usage

Cloud Agents operate on feature branches prefixed with `cursor/` and open
pull requests against `main`. Use this repository to try out agent-driven
changes safely.
